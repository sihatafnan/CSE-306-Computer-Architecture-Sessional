# CSE 306 - Computer Architecture (Sessional)

Coursework for the CSE 306 Computer Architecture sessional (lab). The
assignments implement and simulate fundamental processor building blocks — an
ALU, an IEEE floating-point adder, and the MIPS architecture (single-cycle and
pipelined) — primarily as digital logic circuits.

## Assignments

| Folder | Description |
| ------ | ----------- |
| `Assignment_1_ALU_Simulation/` | Design and simulation of an Arithmetic Logic Unit (ALU). |
| `Assignment_2_FP Adder/` | Floating-point adder implementing IEEE-style addition. |
| `Assignment_3_MIPS/` | Simulation of the MIPS processor architecture. |
| `Assignment_MIPS_Piepeline/` | Pipelined MIPS processor design. |
| `Quiz Exam CSE 306.pdf` | Course quiz material. |

## Tools

The processor/datapath circuits are built and simulated as digital logic
designs. Open the circuit files in each assignment folder with the matching
logic-simulation tool to run them.
