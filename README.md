# CSE 306 - Computer Architecture (Sessional)

Coursework for the CSE 306 Computer Architecture sessional (lab). The
assignments implement and simulate fundamental processor building blocks — an
ALU, an IEEE floating-point adder, and the MIPS architecture (single-cycle and
pipelined) — built as digital logic circuits in **Logisim** (`.circ` files).

## Assignments

| Folder | Description |
| ------ | ----------- |
| `Assignment_1_ALU_Simulation/` | Design and simulation of an Arithmetic Logic Unit (ALU). Contains the block diagram and full circuit (`block_diagram.circ`, `ckt_diagram.circ`) plus a `report.pdf`. |
| `Assignment_2_FP Adder/` | Floating-point adder implementing IEEE-style addition. |
| `Assignment_3_MIPS/` | Single-cycle MIPS processor circuit (`B2_Group3.circ`) with an 8-bit ALU subcircuit, opcode reference (`OPCodes.pdf`), and a `Solution/` folder. |
| `Assignment_MIPS_Piepeline/` | Pipelined MIPS processor design. |
| `Quiz Exam CSE 306.pdf` | Course quiz material. |

## Tools

The processor/datapath circuits are built and simulated as digital logic
designs in [Logisim](https://www.cburch.com/logisim/). Open the `.circ` files in
each assignment folder with Logisim (or a compatible fork such as
Logisim-evolution) to inspect and run them.

## Usage

1. Install Logisim (or Logisim-evolution).
2. Open the `.circ` file from the relevant assignment folder.
3. Drive the inputs and run the simulation to observe the datapath/ALU behavior;
   see each assignment's accompanying PDF for the problem statement and notes.
